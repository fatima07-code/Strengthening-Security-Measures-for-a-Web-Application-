# Strengthening-Security-Measures-for-a-Web-Application-
MERN Authentication Security Enhancement Project
Overview

This project was completed as part of a Cybersecurity Internship task focused on identifying and mitigating common web application vulnerabilities. A MERN Authentication System was analyzed, tested, and improved using various security practices and tools.

Features
User Registration and Login
Password Reset Functionality
JWT-Based Authentication
MongoDB Database Integration
Secure Password Hashing with bcrypt
Input Validation using Validator
HTTP Header Security using Helmet
Application Logging using Winston
Security Assessment Performed
Week 1 – Security Assessment
Explored application functionality
Tested for Cross-Site Scripting (XSS)
Tested basic SQL Injection payloads
Reviewed authentication and password security
Identified security improvement areas
Week 2 – Security Improvements
Added email validation using Validator
Verified password hashing using bcrypt
Implemented JWT authentication
Added Helmet.js for secure HTTP headers
Week 3 – Advanced Security
Performed basic penetration testing using Nmap
Implemented security logging using Winston
Created a security best-practices checklist
